Detailed Description

Linkage Disequilibrium computation class. Will perform diagonal or full model by calling process(). Will perform LD between specified pair of SNPs as well.

Author:
Richard T. Guy (in present form)
Joshua Grab, Matt Steigert, Carl D. Langefeld
