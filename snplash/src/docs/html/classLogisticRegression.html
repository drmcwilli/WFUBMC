Compute the likelhood ratio between null model (beta1) and non-null model (beta2)

D = -2 * sum ( y_i ln (pi_i / y_i) + (1 - y_i) ln ((1-pi_i) / (1 - y_i) ) ) for given model pi.

Compute G = D model 1 - D model 2

Returns:
The likelihood statistic.
