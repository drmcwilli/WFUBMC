Create a string with the 'number' embedded in 'numspaces' spaces.

Doubles can satisfy several criteria: very closeto zero, requiring scientific notation. have digits on either side of the decimal.

Parameters:
number To be placed
numspaces Number of slots alloted
precision The desired numerical precision to be printed.
buffer Number of spaces to print ahead of the number.
Returns:
std::string A string with the correct embedded number.

Sometimes prints with too many spaces if a large exponent is involved.
